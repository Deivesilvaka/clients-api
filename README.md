## Installation

```bash
$ npm install
```

O arquivo .env pode ser criado igual ao .env.example

## Running the app

Com o docker rodano, basta rodar os seguintes comandos.

```bash
# docker
$ docker compose up -d

# build
$ nest build

# migrations
$ npm run migration:run

# run application
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test
```
